## 3. COMMON FAILURE MODES (PROVEN)

### ‚ùå Failure #1: Missing noise config in base registration
**Symptom**: Spec map is completely FLAT (solid color). No sparkle, no flake, no texture.
**Root cause**: Base registration only has `"M": 245, "R": 8, "CC": 16` ‚Äî no `noise_scales` or `perlin` config.
**Fix**: Add noise config to registration:
```python
# For sparkle/flake (fine dots):
"noise_scales": [1, 2, 4], "noise_weights": [0.5, 0.3, 0.2], "noise_M": 50, "noise_R": 18

# For flowing/organic (smooth bands):
"perlin": True, "perlin_octaves": 3, "perlin_persistence": 0.5, "perlin_lacunarity": 2.0, "noise_M": 40, "noise_R": 25
```
**Where**: `compose_finish()` at ~line 6640 checks for `noise_scales`/`perlin`/`brush_grain`. If none exist, falls back to `np.full()` ‚Äî completely flat.

### ‚ùå Failure #2: Paint function effects too weak
**Symptom**: Paint layer barely changes color.
**Root cause**: Effect multipliers are tiny (e.g., `0.03 * pm`, `0.06 * pm`).
**Fix**: Use multipliers of `0.15-0.45` for visible effects. Remember `pm` comes in at ~1.5 * 2.5 (boost) = ~3.75 at the function.

### ‚ùå Failure #3: pm used as frequency multiplier (cos/sin wrapping)
**Symptom**: Effect produces random noise instead of smooth patterns.
**Root cause**: Code like `cos(angle * pm * pi * 2.5)` ‚Äî when pm=3.75, the argument hits ¬±26 radians, wrapping cos/sin 4+ times ‚Üí random noise.
**Fix**: Use pm to control **blend strength** (0 to 1.0 cap), NOT frequency:
```python
blend = np.clip(pm * 0.25, 0, 1.0)
# Apply: result * blend * mask + original * (1 - blend * mask)
```

### ‚ùå Failure #4: No _BASE_PAINT_BOOST in pipeline
**Symptom**: Pattern effects visible but base effects invisible.
**Root cause**: `compose_paint_mod()` had `_PAT_PAINT_BOOST = 3.5` for patterns but no equivalent for bases.
**Fix**: Added `_BASE_PAINT_BOOST = 2.5` to both `compose_paint_mod()` and `compose_paint_mod_stacked()`.
**Location**: ~line 7107 and ~line 7195 in `shokker_engine_v2.py`.

### ‚ùå Failure #5: All bases produce identical spec noise
**Symptom**: Every base has the same noise pattern in spec map.
**Root cause**: `compose_finish()` used `seed + 100` for ALL bases.
**Fix**: Added `_base_seed_offset = abs(hash(base_id)) % 10000` to make each base unique.
**Location**: ~line 6614 in `compose_finish()`.

### ‚ùå Failure #6: Missing swatch preview function
**Symptom**: Base/pattern shows a flat color swatch instead of a pattern preview.
**Root cause**: No canvas rendering function in `baseFns` / `patternFns` / `monoFns` in HTML.
**Fix**: Add a canvas function that visually represents the effect.

## 4. BASE REGISTRATION ANATOMY

```python
"base_id": {
    # === REQUIRED ===
    "M": 245,           # Metallic channel (0=none, 255=mirror)
    "R": 8,             # Roughness channel (0=smooth, 255=rough)
    "CC": 16,           # Clearcoat (0=none, 16=perfect clear, 255=degraded)
    "paint_fn": fn,     # Paint layer modifier function
    "desc": "...",      # Description string

    # === SPEC NOISE (pick ONE type) ===
    # Type A: Multi-scale noise (good for sparkle, flake, speckle)
    "noise_scales": [1, 2, 4],       # Pixel scales (lower = finer grain)
    "noise_weights": [0.5, 0.3, 0.2], # Weight per scale
    "noise_M": 50,                    # Metallic variation magnitude
    "noise_R": 18,                    # Roughness variation magnitude

    # Type B: Perlin noise (good for flowing, organic, bands)
    "perlin": True,
    "perlin_octaves": 3,              # More = more detail
    "perlin_persistence": 0.5,        # Higher = more high-freq detail
    "perlin_lacunarity": 2.0,         # Frequency multiplier between octaves
    "noise_M": 40,
    "noise_R": 25,

    # Type C: Brush grain (good for brushed metal)
    "brush_grain": True,
    "noise_M": 15,
    "noise_R": 30,

    # === OPTIONAL ===
    "noise_CC": 10,     # Clearcoat spatial variation (rarely needed)
}
```

## 6. PAINT FUNCTION TEMPLATE

```python
def paint_effect_name(paint, shape, mask, seed, pm, bb):
    """One-line description of the visual effect."""
    h, w = shape

    # pm comes in at ~3.75 (1.5 * _BASE_PAINT_BOOST=2.5)
    # Use pm to control blend strength, NOT frequency!
    blend = np.clip(pm * 0.25, 0, 1.0)

    # --- Color modification ---
    r, g, b = paint[:,:,0].copy(), paint[:,:,1].copy(), paint[:,:,2].copy()
    # ... compute new colors ...
    paint[:,:,0] = new_r * blend * mask + r * (1 - blend * mask)

    # --- Optional: sparkle layer ---
    rng = np.random.RandomState(seed + UNIQUE_OFFSET)  # Use unique offset!
    sparkle = rng.random((h, w)).astype(np.float32)
    sparkle = np.where(sparkle > 0.90, (sparkle - 0.90) * 2.5, 0)
    paint = np.clip(paint + sparkle[:,:,np.newaxis] * mask[:,:,np.newaxis], 0, 1)

    # --- Brightness boost ---
    paint = np.clip(paint + bb * 0.6 * mask[:,:,np.newaxis], 0, 1)
    return paint
```

## 7. SWATCH PREVIEW FUNCTION TEMPLATE

Located in `paint-booth-v2.html` in the `baseFns`, `patternFns`, or `monoFns` objects.

```javascript
effect_id: (ctx, w, h) => {
    // Background color/gradient
    const g = ctx.createLinearGradient(0, 0, w, h);
    g.addColorStop(0, '#color1'); g.addColorStop(1, '#color2');
    ctx.fillStyle = g; ctx.fillRect(0, 0, w, h);

    // Pixel-level effects (sparkle, noise, patterns)
    const rng = _seededRng(UNIQUE_SEED);
    const id = ctx.getImageData(0, 0, w, h);
    for (let i = 0; i < id.data.length; i += 4) {
        if (rng() > 0.90) {
            id.data[i] += 40;     // R
            id.data[i+1] += 30;   // G
            id.data[i+2] += 20;   // B
        }
    }
    ctx.putImageData(id, 0, 0);
},
```

**Helper functions available**: `_seededRng(seed)`, `_hslToRgb(h,s,l)`, `_simpleNoise2D(w,h,scales,weights,rng)`

## 9. PIPELINE CONSTANTS

```
_INTENSITY_SCALE["paint"]  = 1.50   (pm = user_slider * 1.50)
_INTENSITY_SCALE["spec"]   = 2.00   (sm = user_slider * 2.00)
_INTENSITY_SCALE["bright"] = 0.15   (bb = user_slider * 0.15)

_BASE_PAINT_BOOST = 2.5            (bases get pm * 2.5)
_PAT_PAINT_BOOST  = 3.5            (patterns get pm * 3.5)

Effective pm at paint_fn: ~1.0 * 1.50 * 2.5 = 3.75 (no pattern)
Effective pm at paint_fn: ~1.0 * 1.50 * 2.5 * 0.7 = 2.625 (with pattern)
```

## 10. CHECKLIST FOR FIXING A BASE GROUP

1. **Find registrations**: `grep "base_id"` in `shokker_engine_v2.py` and `shokker_24k_expansion.py`
2. **Check noise config**: Does it have `noise_scales`/`perlin`/`brush_grain`? If not ‚Üí **Failure #1**
3. **Check paint_fn**: View the function. Are multipliers > 0.10? Does it use pm as blend not frequency?
4. **Check swatch preview**: Search `baseFns` / `monoFns` in HTML. Does the ID have a canvas function?
5. **Choose noise type**: Use the guide in Section 5 above
6. **Test render**: Verify both paint layer AND spec map show visible effects
7. **Deploy**: Restart server, hard refresh browser (Ctrl+Shift+R)
